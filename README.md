> **Academic Project** — DIC2 Cybersécurité, École Supérieure Polytechnique de Dakar (ESP)

## 📌 Overview

Designed and deployed a **complete Network Security Monitoring (NSM) lab** simulating a real SOC environment. The infrastructure captures, stores and analyzes all network traffic using a hybrid approach: full packet capture (PCAP) + aggregated flows (NetFlow) + real-time intrusion detection.
